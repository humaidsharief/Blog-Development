# Blog-Development

## Description
Blog site - Simple blogging website where users can login and post whatever they want to

Functions:
- Post blogs to everyone around the world
- Login + Registration page
- Actively updating
- Like posts
- Delete posts

## Technologies
- Python 3
- SQlite3
- Flask
- Werkzeug Security
- HTML/CSS
- Jinja2

## Installation and Launch
1. **Clone the Repository**
2. **Create a virtual environment**
3. **Install dependencies:**

   SQlite3,
   Flask,
   Jinja2,
   Werkzeug Security,
4. **Run the program**
